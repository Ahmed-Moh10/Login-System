# 🔐 Smart Login System

A simple and responsive login system built using **HTML, CSS, Bootstrap, and JavaScript**.
The project allows users to sign up, log in, and see a personalized welcome page.

---

## 🚀 Features

* ✅ User Registration (Sign Up)
* ✅ User Login (Sign In)
* ✅ Input Validation
* ✅ Error Handling Messages
* ✅ Local Storage for saving user data
* ✅ Logout Functionality
* ✅ Fully Responsive Design (Mobile, Tablet, Desktop)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* JavaScript (Vanilla JS)
